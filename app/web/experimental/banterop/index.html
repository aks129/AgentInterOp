Help & Resources

How to Use

  • Select a Scenario: Choose a preset scenario (e.g., "Breast Cancer Screening") or enter a custom URL in the Configuration panel.
  • Load Scenario: Click "Load Scenario" to fetch the scenario details.
  • Start Conversation: In the Conversation panel, click "Start New Conversation".
  • Chat: Use the message input to interact with the agent.

FAQ

What is Banterop V2?
Banterop V2 is a platform for testing and demonstrating agent interoperability protocols, specifically focusing on healthcare scenarios.
Why is the API unavailable?
If the status indicator is offline, ensure the backend server is running (python -m uvicorn app.main:app).
Can I use my own agents?
Yes, you can provide a custom Agent Card URL in the Agents tab to connect to external agents.
