Agent Management Help

How to Use

Creating an Agent
Click "+ Create New Agent" to define a new agent. You'll need to specify its role, domain, and constitutional guidelines (constraints, ethics, capabilities).
Managing Agents
Use the "Activate/Deactivate" buttons to control agent availability. "Archive" removes an agent from the active list.

FAQ

What is the ADK Framework?
The Agent Development Kit (ADK) provides a structured way to define agent behaviors, ensuring they adhere to specific constraints and ethical guidelines.
What are Templates?
Templates are pre-configured agent definitions that you can use as a starting point for common healthcare roles.
